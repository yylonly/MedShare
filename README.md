MedShare: Medical Resource Sharing among Autonomous Healthcare Providers

## Pre-requirements
* Docker CE
* Docker Compose

## Install
* git clone https://github.com/yylonly/medshare.git
* cd medshare
* docker-compose build
* docker-compose up -d
